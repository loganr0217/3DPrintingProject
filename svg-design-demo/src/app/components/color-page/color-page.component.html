Window Designer

Add each panel that creates a window from top to bottom. To get the d attribute for a panel, open it in InkScape and select Edit -> XML Editor: a box will appear on the right of your window. If you click on the panel, it should highlight the corresponding path. Sometimes, it will select an outer layer: if this happens, just look under that layer in the XML editor and the path will be within that.
